**코드규칙**
전역변수 [ g_변수명 ]
멤버변수 [ m_변수명 ]
매개변수 [ _변수명  ]

**변수명 작성 규칙
1. 약어 금지 (영단어 풀스펠링 다 써야함 ex. count -> cnt, point -> pt )
2. 변수명 작성할때 단어 생각안나면 영어사전 찾아서 작성하고 주석으로 뜻 설명.
   (ex. int property = 10 ; //속성 )
3. 여러단어의 경우에는 단어의 첫글자는 대문자 (ex. int StudentNumber; )
4. 변수명은 용도에 맞는 단어와 자료형을 선택해서 작성.
   예를 들어, 2가지 경우만 체크하면 된다면 가급적 bool형으로 작성하고 무엇을 체크해야하는지
   변수명으로 용도를 나타낸다.
5. 대문자로만 작성할 때 단어간 구분이 필요하면 _로 띄어쓰기 할 것
   (예. #define TIME_MANAGER ....)

** 코드작성 규칙
1. 대괄호 { }는 무조건 줄바꿔서 ( ) 아랫줄에 내려서 작성
2. 띄어쓰기는 자동완성 되는대로
